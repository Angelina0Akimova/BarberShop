📌 BarberShop — WPF Desktop Application
BarberShop — десктопное приложение для управления барбершопом, разработанное на C# с использованием WPF (Windows Presentation Foundation) и .NET Framework в среде Visual Studio в рамках дипломной работы.

---

🧾 Описание
Это WPF-приложение предназначено для автоматизации процессов барбершопа: управление услугами, мастерами, клиентами и записями на стрижки/услуги. UI создан с использованием XAML, бизнес-логика реализована на C#.

---

🛠Технологии

Проект построен с использованием:

* 💻 C# (.NET Framework)
* 📊 WPF (Windows Presentation Foundation)
* 🛠 Visual Studio (IDE)
* 📦 MVVM-паттерн для организации кода
* 💾 Подключение к базе данных SQL

---

🚀 Возможности

Регистрация и авторизация пользователей
Просмотр списка услуг
Управление мастерами (CRUD)
Планирование записи клиентов
История посещений
Фильтрация и поиск данных

📦 Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/Angelina0Akimova/BarberShop.git
   ```
2. Открой решение (`.sln`) в **Visual Studio**.
3. Восстанови NuGet-пакеты, если требуется.
4. Настрой подключение к базе данных .
5. Запусти проект через `F5` или Start.

---

📌 Использование

После запуска приложения:

1. Ввод логина/пароля.
2. Откроется главное окно с меню навигации.
3. Через UI можно просматривать, добавлять, редактировать и удалять данные: услуги, мастеров, записи и т.д.
